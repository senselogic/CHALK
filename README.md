# Chalk

Pastel color theme for Geany and Sublime Text.

## Installation

For Geany :

```bash
cp chalk_theme.conf ~/.config/geany/colorschemes/
```

For Sublime Text :

```bash
cp chalk_theme.tmTheme ~/.config/sublime-text-3/Packages/User/
```
